Arcade Workshop & AI Guide

The Arcade Workshop is a powerful environment for creating, editing, and publishing mini-games. Integrated with a local AI Companion, you can build entire experiences using natural language.

The AI Companion

The AI Chatbot in the Arcade isn't just for conversation—it's a system-aware assistant. It can see your active workshop project, analyze your code, and perform file operations on your behalf.

  • Context Awareness: The AI knows which game you are editing and can read all its files.
  • Surgical Edits: It can perform line-by-line patching to fix bugs or add features.
  • Full Rewrites: For major changes, it can replace entire files with optimized versions.

Workshop Slash Commands

Use these commands in the Arcade Chat to trigger specific automation workflows:

/edit Request specific code changes to your active workshop file.
/fix Automatically analyze the active file for bugs and apply surgical patches.
/publish Prepare and publish your current workshop project to the global library.
/clear Reset the chat context and clear the conversation history.
/help Display a list of all available commands and their usage.

System & Media Control

The AI Companion can also bridge with your Windows PC to control media playback. Try asking the AI to perform these actions:

  • Spotify Control: "Play some music on Spotify", "Next track", or "Pause Spotify".
  • YouTube Control: "Play the YouTube video", "Previous video", or "Set volume to 50%".
  • Status Query: "What is playing right now?" or "Is the desktop bridge active?"

*Note: These commands require the Signal Share Desktop Bridge to be running on your host machine.

Publishing to the Library

When you are happy with your game, you can share it with the community:

  1. Ensure your game has an index.html entry point.
  2. Add a description and tags in the Workshop Edit panel.
  3. Use the Publish button or the /publish command.
  4. Once published, your game will appear in the Workshop tab for others to play.
  5. Leaderboards: The system automatically detects score-related code and creates global rankings for your game.
